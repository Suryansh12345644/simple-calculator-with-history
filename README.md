# Simple Calculator with History

## Overview
A command-line based calculator built using Python that performs basic arithmetic operations and stores the history of calculations for later review.

## Features
- Addition, subtraction, multiplication, and division
- Stores calculation history locally
- View previous calculations during runtime
- Lightweight and beginner-friendly

## Tech Stack
- Python 3

## How to Run
1. Clone the repository
2. Navigate to the project directory
3. Run the program using:
   python calculator.py

## Use Case
This project is designed to strengthen fundamentals of Python programming, including file handling, control flow, and user input processing.
